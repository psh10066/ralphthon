## Iteration 2 - Phase 2~6: 코어 엔진 + 화면 + 텔레그램 ✅

### 완료 사항
- POST /api/analyze: Claude Vision API, 1~5장 유연, 가설 질문, 6축+팔레트
- POST /api/chat: 4타입 분기, 링크 3단계 폴백(cheerio→AI리서치→붙여넣기), 인사이트 감지
- GET /api/profile: 에센스+인사이트+클러스터+키워드
- GET /api/sessions: 세션 목록+인사이트 카운트
- POST /api/telegram/webhook: 온보딩, 사진→에센스, 대화, 인사이트→웹링크, /start WEB_ 연결
- POST /api/db/init: Supabase 스키마 초기화
- 10개 페이지 UI: 랜딩, 업로드, 로딩, 에센스(WOW), 홈, 대화, 세션상세, 세션로그, 프로필
- WOW 화면: 가설 질문 + 반응 버튼 3개 (맞아요/좀 다른데/잘 모르겠어요) + 분기
- InputArea: drop 버튼, 마이크(STT), 사진, 링크
- 빈 상태 처리: 에센스 미완 안내, 인사이트 0개, 패턴 생성 중
