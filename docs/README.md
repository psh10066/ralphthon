## 한 줄

사진, 글, 메모 — 뭐든 넣으면 AI가 호기심으로 질문하고, 당신이 몰랐던 당신을 언어로 돌려준다. 쌓일수록 "그때의 나"와 "지금의 나"가 보인다.

---

## 구조

```
대화는 어디서든 시작한다 (텔레그램, 웹, 어디든)
앱은 그게 잘 쌓이는 곳이다
```

```
[입력]  사진 / 글 / 링크 / 메모 / 음성 — 어떤 채널에서든
   ↓
[대화]  AI가 호기심으로 질문한다 ("왜 이걸 올렸어?")
   ↓
[발견]  글의 언어가 나의 언어로 바뀐다
   ↓
[축적]  지형도처럼 쌓인다 — 어떤 영역이 두꺼운지 보인다
   ↓
[시간]  "그때의 나"와 "지금의 나"가 다르다는 걸 안다
```

## 핵심 결정

| 결정 | 이유 |
|------|------|
| 입구 = 호기심, 깊이 = FOMO 해소 | FOMO와 사진은 다른 감정. 섞으면 퍼널이 꼬임 |
| 에센스 = 결론이 아니라 대화의 시작 | "이렇게 읽어봤어요, 더 알고 싶어요" 태도. 진단 금지 |
| 첫 질문 = 열린 호기심, 가설은 2턴부터 | 감각→심리상태 점프는 진단. "뭐가 끌렸어요?"가 먼저 |
| 메타대화 = 모든 입력의 즉시 반응 | 별도 단계가 아니라, 뭘 넣든 바로 질문이 시작 |
| wow = 에센스 언어, UI 테마 아님 | 테스트 결과 시각적 차이보다 언어적 해석이 강력 |
| 공유 = 대화 후, 발견이 생긴 뒤 | 에센스 결과에서 바로 공유하면 "결론"이 됨 |
| 텔레그램 = 대화 시작점, 웹 = 축적 | 대화는 어디서든, 축적은 한 곳에서 |
| 지형도 = 축적 시각화 | 레이어링. 어떤 영역이 두꺼운지 한눈에 |
| 시간 축 = "그때의 나" | 현재 상태만이 아니라 변화사가 가치 |
