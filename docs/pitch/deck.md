## Slide 2. 문제

### 사람들은 자기가 뭘 좋아하는지는 안다. 왜 좋아하는지는 모른다.

핀터레스트 200장. 인스타 저장 47개. 카메라롤 3,000장.

다 "나"인데 — 아무도 그걸 읽어주지 않았다. 본인도 읽을 줄 모른다.

- 옷을 사면 매번 비슷한 걸 고르면서 "나는 왜 이런 걸 좋아하지?"를 한 번도 안 물어봤다
- 핀터레스트에 200장 저장 — 다 "나"인데 아무도 읽어주지 않았다
- "나를 소개해주세요"에서 막힌다 — MBTI 말고 할 말이 없다

자기소개를 못 하는 게 아니라, **자기 이해가 없는 거다.**

## Slide 3. 인사이트

### 좋아하는 것들이 이미 당신을 말하고 있었다.

같은 사진을 봐도 사람마다 끌리는 게 다르다. 그 끌림에는 패턴이 있다.

끌린 것들의 교집합에 **"지금의 나"**가 있다.

근데 대부분은 이걸 모른다. "내가 왜 이걸 좋아하지?"를 물어본 적이 없으니까. 물어봐도 혼자서는 답하기 어렵다.

## Slide 4. 솔루션 — 데모 흐름

### 읽어보고, 물어보는 것. 진단이 아니라 거울.

```
사진 5장 업로드
    ↓
AI가 에센스를 읽어본다
    ↓
6축 분석 + 한 줄 에센스 + 열린 질문 → 대화 시작
```

**6축 분석**: 양감 / 질감 / 투명도 / 촉각 / 무게 / 온도

사진에서 색감이나 구도만 보는 게 아니다. "이 사진이 왜 당신을 건들였는지"를 읽어본다.

결과물:
- **한 줄 에센스**: "손 닿는 곳의 아늑함을 모으는 사람"
- **관찰**: "사진이 전부 가까이에 있는 것들이거든요."
- **열린 질문**: "뭐가 끌렸어요?"

진단하지 않는다. **"이렇게 읽어봤는데, 더 알고 싶어요"**라고 한다. 에센스는 결론이 아니라 대화의 시작이다.

## Slide 5. 깊이 — 글 읽기

### "요즘 저장만 해둔 글 있어요?"

글을 넣으면 요약이 오는 게 아니라 **질문이 온다.**

- 글: "속도를 늦춰야 빨라진다"
- AI: "이 문장에 걸렸다면, 지금 뭘 빠르게 하려다 막혀있어?"

글의 언어가 나의 언어로 바뀐다:
- 글: "Closer 근육이 필요하다"
- 나: "준비 안 된 채로 꺼내는 연습이 필요해."

그리고 이 발견이 에센스와 연결된다. 사진에서 읽은 "나"와, 글에서 발견한 "나"가 하나의 프로필로 쌓인다.

## Slide 6. 축적 = 해자

### 1회는 대체 가능. 30회 축적은 대체 불가능.

| | MBTI | ChatGPT | 취향 거울 |
|---|---|---|---|
| **입력** | 선택지 | 텍스트 | 사진 + 글 |
| **결과** | 4글자 | 요약 | 에센스 + 질문 |
| **10회 후** | 같은 결과 | 리셋 | 패턴이 보이기 시작 |
| **30회 후** | 같은 결과 | 불가능 | **변화사가 존재** |

MBTI는 매번 같은 결과를 준다. ChatGPT는 매번 리셋된다. 여기는 축적된다.

- "지난달에도 '통제'에 대한 글에서 반응했는데, 이번엔 다른 각도네요"
- "취향이 조금 바뀌었어요 — 6개월 전엔 미니멀했는데 요즘은 따뜻한 쪽으로"

**"나의 변화사"가 존재하는 것.** 이게 moat다.

## Slide 7. 검증

### 28세션, 2개월. 직접 썼다.

메타대화 코어를 본인이 2개월간 사용하며 검증했다.

**실제 전환 사례 3가지:**

1. "속도를 늦춰야 빨라진다" → "준비 안 된 채로 꺼내는 연습이 필요해"
   - 남의 글이 나의 언어로 바뀐 순간
2. flowkater 글 3편 → NSM 전제 전환
   - "다운로드 늘리자"에서 "개입을 줄이자"로. 글이 촉매, 발견은 안에서 일어남
3. 볼타 CLI 리뷰 → "경비 관리 도구가 아니라 CFO 재무 인프라"
   - 외부 자극이 내 일의 프레임을 바꿈

**동료 반응:** "일에서 오는 힘듦을 해소하고 있다"

글은 촉매였고, 발견은 나 안에서 일어났다.

## Slide 8. 기술

### Claude Vision + Text API. 소크라틱 루프.

```
사진 5장
    ↓
[Claude Vision] 이미지 분석
  6축: 양감 / 질감 / 투명도 / 촉각 / 무게 / 온도
    ↓
[LLM] 에센스 합성 + 가설 생성
  소크라틱 루프로 모호성 제거
    ↓
프로필 생성 → 대화 → 축적
```

**AI 자율성**: 사진 분석 → 프로필 → 대화까지 사람 개입 0

**모호성 제거**: "이 사진에서 끌리는 건 색감인가, 구도인가, 분위기인가?"를 AI 내부에서 반복 검증한 뒤 사용자에게 건넨다.
