### BUG-008: 주간 리포트 — TC만 누락

bugfix-requirements.md에 기능 정의는 있으나 test-cases.md에 TC가 없음.

**추가할 테스트케이스:**

| # | 시나리오 | 조작 | 기대 결과 |
|---|---------|------|----------|
| 1 | 리포트 생성 조건 | 세션 1개 이상 있는 주 → 프로필 방문 | "이번 주" 리포트 카드 표시 |
| 2 | 세션 없는 주 | 대화 0건인 주 → 프로필 방문 | 리포트 카드 없음 또는 "이번 주는 대화가 없었어요" |
| 3 | 리포트 내용 | 리포트 카드 탭 | 대화 횟수 + 새 발견 + 반복 키워드 + 에센스 변화 + 한 줄 요약 |

### BUG-009: 발견 & 키워드 축적 안 됨 — 완전 누락

**bugfix-requirements.md 추가 내용:**

- 현상: 대화에서 인사이트 카드 `[ 기록할까요? ]`를 탭해도 프로필의 "축적된 발견"에 반영 안 됨. 반복 키워드도 빈 상태.
- 기대 동작:
  - 인사이트 카드 `[ 기록할까요? ]` → Supabase 저장 → 프로필 "축적된 발견"에 즉시 반영
  - 대화 종료 시 AI 응답에서 태그 추출 → "반복 키워드" 섹션 갱신
  - 발견 0건일 때: "아직 발견이 쌓이지 않았어요" 빈 상태 표시
- 우선순위: medium (프로필 기능이지만 축적이 서비스 해자)

**추가할 테스트케이스:**

| # | 시나리오 | 조작 | 기대 결과 |
|---|---------|------|----------|
| 1 | 발견 저장 | 채팅 중 인사이트 카드 `[ 기록할까요? ]` 탭 | 카드 "저장됨 ✓" + 프로필 "축적된 발견"에 추가 |
| 2 | 발견 빈 상태 | 발견 0건 → 프로필 방문 | "아직 발견이 쌓이지 않았어요" 표시 |
| 3 | 발견 누적 | 3개 세션에서 각 1개 발견 저장 | 프로필에 3개 발견 표시, 최신순 |
| 4 | 키워드 추출 | 대화 3건 후 프로필 방문 | 반복 키워드 섹션에 태그 표시 |
| 5 | 키워드 빈 상태 | 대화 0건 → 프로필 방문 | 반복 키워드 섹션 비어있거나 숨김 |
| 6 | 넘어가기 | 인사이트 카드 `[ 넘어가기 ]` 탭 | 프로필에 반영 안 됨 |

### BUG-010: 세션 로그 조회 불가 — 완전 누락

**bugfix-requirements.md 추가 내용:**

- 현상: `/sessions` 화면에서 과거 세션 목록이 표시되지 않음
- 기대 동작:
  - 완료된 세션이 날짜별로 목록 표시
  - 각 세션: 날짜 + 요약 한 줄 + 발견 개수
  - 탭 시 `/chat/:sessionId`로 이동 (읽기 전용)
  - 세션 0건: "아직 대화가 없어요" 빈 상태
- 우선순위: medium

**추가할 테스트케이스:**

| # | 시나리오 | 조작 | 기대 결과 |
|---|---------|------|----------|
| 1 | 세션 목록 | 대화 3건 후 세션 탭 | 3개 세션 날짜순 표시 |
| 2 | 빈 상태 | 대화 0건 → 세션 탭 | "아직 대화가 없어요" 표시 |
| 3 | 세션 상세 | 세션 카드 탭 | `/chat/:sessionId`로 이동, 대화 원문 표시 |
| 4 | 읽기 전용 | 세션 상세에서 입력 시도 | 입력 영역 없음 (읽기 전용) |
| 5 | 발견 표시 | 발견 1개 있는 세션 상세 | "이 세션에서 나온 발견" 섹션에 표시 |
