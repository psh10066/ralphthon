## 카테고리: 사진 업로드

### BUG-001: 사진 일괄 선택 불가
- 위치: 사진 입력 화면
- 현재 동작: 사진을 한 장씩만 선택 가능
- 기대 동작: 여러 장을 한번에 선택하여 업로드 가능해야 함
- 우선순위: high

### BUG-002: 사진 용량 제한 없음
- 위치: 사진 업로드 기능
- 현재 동작: 용량 제한 없이 어떤 크기의 파일이든 업로드 가능
- 기대 동작: 파일 크기 상한(예: 10MB)을 두고, 초과 시 안내 메시지 표시
- 우선순위: high

## 카테고리: 채팅 (대화창)

### BUG-003: 결과 화면에서 첫 채팅 후 입력 잠김
- 위치: 사진 결과 화면 > "어디가 다르다고 느꼈어요?" 응답 버튼 클릭 후
- 현재 동작: 첫 번째 메시지 전송 후, 두 번째 메시지를 한 번 더 보내야 채팅이 활성화됨
- 기대 동작: 첫 번째 메시지 전송 후 즉시 연속 채팅 가능
- 우선순위: high

### BUG-004: 사진 입력 시 이후 대화 불가
- 위치: 대화창에서 이미지 전송 시
- 현재 동작: 채팅 중 사진을 보내면 이후 텍스트 입력이 불가능해짐
- 기대 동작: 사진 전송 후에도 정상적으로 텍스트 채팅 가능
- 우선순위: high

### BUG-005: 탭 이동 후 채팅 세션 초기화
- 위치: 하단 네비게이션 탭
- 현재 동작: 다른 탭으로 이동했다가 채팅 탭으로 돌아오면 기존 대화 세션이 사라짐
- 기대 동작: 탭 전환 후에도 진행 중인 채팅 세션이 유지되어야 함
- 우선순위: high

## 카테고리: AI 대화 품질 (프롬프트)

### BUG-006: 오타 지적 톤이 부정적
- 위치: AI 채팅 응답
- 현재 동작: 사용자의 오타를 지적할 때 비꼬거나 꼽주는 듯한 톤
- 기대 동작: 오타 지적 시 부드럽고 친근한 톤 사용, 또는 오타 지적 자체를 하지 않음
- 우선순위: medium

### BUG-007: 질문만 반복하여 대화가 루즈함
- 위치: AI 채팅 응답
- 현재 동작: AI가 계속 질문만 하여 대화 진행이 느리고 지루함
- 기대 동작: 질문 사이에 인사이트, 공감, 피드백을 적절히 섞어 대화에 밀도감 부여
- 우선순위: medium

## 카테고리: 프로필

### BUG-008: 주간 리포트 생성 불가
- 위치: 프로필 화면 > 주간 리포트
- 현재 동작: 주간 리포트가 생성되지 않음
- 기대 동작: 축적된 대화 데이터를 기반으로 주간 리포트가 정상 생성
- 우선순위: medium

### BUG-009: 축적된 발견 & 반복 키워드가 쌓이지 않음
- 위치: 프로필 화면 > 축적된 발견 & 반복 키워드 섹션
- 현재 동작: 대화를 진행해도 해당 섹션에 데이터가 쌓이지 않음. 쌓이는 조건도 불명확
- 기대 동작: 대화 세션 완료 시 키워드와 발견 사항이 자동으로 축적되어 표시
- 우선순위: medium

## 카테고리: 세션

### BUG-010: 세션 로그 조회 불가
- 위치: 세션 화면
- 현재 동작: 과거 세션 로그가 표시되지 않음
- 기대 동작: 완료된 세션의 대화 로그가 목록으로 조회 가능
- 우선순위: medium

## 카테고리: 컨셉/구조 변경 요구사항

### REQ-001: "거울" 컨셉 제거
- 현재 동작: 6축 비주얼 분석이 wow point라는 구조로 설계됨
- 기대 동작: 거울 컨셉 및 6축 비주얼 분석 구조 전면 제거
- 우선순위: high

### REQ-002: "지형도" 컨셉 제거
- 현재 동작: 등고선 맵 시각화가 포함되어 있음
- 기대 동작: 등고선 맵 시각화 전면 제거
- 우선순위: high

### REQ-003: 입력을 사진 5장으로 한정하지 않음
- 현재 동작: 사진 5장 고정 입력
- 기대 동작: 사진 1장, 글, 링크 등 어떤 형태든 하나만 drop하면 동작
- 우선순위: high

### REQ-004: 비주얼만 분석하지 않음
- 현재 동작: 사진의 비주얼(색감, 구도 등)만 분석
- 기대 동작: 사진의 내용 + 무드를 함께 읽되, 메타대화 스킬은 그대로 유지
- 우선순위: high

### REQ-005: 대화를 강제하지 않음
- 현재 동작: 분석 후 반드시 대화를 진행해야 하는 흐름
- 기대 동작: 분석 결과를 던져주고, 사용자가 반응하면 대화 진행 / 안 하면 그냥 쌓임
- 우선순위: high

### REQ-006: 축적 방식 변경 — 세션 저장이 아닌 주제별 클러스터링
- 현재 동작: 축적 = 세션 단위 저장
- 기대 동작: 세션 단위가 아닌 주제별로 클러스터링하여 축적
- 우선순위: high
