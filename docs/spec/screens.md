# 랄프톤 — 화면 설계

디자인 톤: 투명, 유리, 여백, 거울. 군더더기 없음. 콘텐츠가 장식.

모바일 퍼스트 375px, 데스크톱 max-width 480px 센터 정렬.

---

## `/` — 원페이지 퍼널

### 랜딩 (뷰포트 1)

```
┌──────────────────────────────────────┐
│                                      │
│     사진 5장이면                      │
│     당신이 보입니다.                  │
│                                      │
│         [ 시작하기 ]                  │
│                                      │
│  ┌────────┐ ┌────────┐ ┌────────┐   │
│  │ 예시1  │ │ 예시2  │ │ 예시3  │   │
│  │ 에센스 │ │ 에센스 │ │ 에센스 │   │
│  │ 카드   │ │ 카드   │ │ 카드   │   │
│  └────────┘ └────────┘ └────────┘   │
│       ← 가로 스크롤 →                │
└──────────────────────────────────────┘
```

### 업로드 (뷰포트 2)

```
┌──────────────────────────────────────┐
│                                      │
│   끌리는 사진으로.                    │
│   예쁠 필요 없어요.                   │
│                                      │
│   ┌─────┐ ┌─────┐ ┌─────┐          │
│   │  +  │ │  +  │ │  +  │          │
│   └─────┘ └─────┘ └─────┘          │
│       ┌─────┐ ┌─────┐              │
│       │  +  │ │  +  │     0/5      │
│       └─────┘ └─────┘              │
│                                      │
│        [ 나를 읽어줘 ]                │
└──────────────────────────────────────┘
```

5장 채워야 CTA 활성화. 모바일: 카메라롤 직접 접근.

### 로딩 → 거울 → 대화 전환

로딩 완료 후 에센스 결과가 나오면, 하단에 관찰 + 열린 질문이 표시된다.
사용자가 자유 응답을 입력하면 → 즉시 `/chat`으로 전환.
에센스 결과 화면은 "끝"이 아니라 "첫 대화 턴"이다.

### 로딩

```
┌──────────────────────────────────────┐
│                                      │
│         당신을 읽고 있어요...         │
│                                      │
│   ✓ 색감을 보고 있어요               │
│   ✓ 질감을 느끼고 있어요             │
│   ● 온도를 재고 있어요               │
│   ○ 언어로 옮기고 있어요             │
│                                      │
└──────────────────────────────────────┘
```

4단계 각 3초. ✓ 완료, ● 현재, ○ 대기.

### 거울 — 에센스 결과 (wow → 첫 대화)

```
┌──────────────────────────────────────┐
│   ┌────────────────────────────┐     │
│   │   (업로드한 사진 5장 strip) │     │
│   └────────────────────────────┘     │
│                                      │
│   사진에서 이렇게 읽어봤어요.          │
│                                      │
│   "손 닿는 곳의 아늑함을              │
│    모으는 사람"                        │
│                                      │
│   양감    채워진 둥근 볼륨             │
│   질감    도자기 유약, 나무결          │
│   투명도  불투명. 빛을 품는 물질들     │
│   촉각    따뜻하다                    │
│   무게    탁자 위에 올려진 존재감      │
│   온도    미지근 ~ 따뜻               │
│                                      │
│   ● ● ● ● ●  (컬러 팔레트)          │
│                                      │
│   ─── 투명한 구분선 ───               │
│                                      │
│   이게 다는 아닐 수 있어요.            │
│   더 알고 싶어서 하나 물어볼게요.      │
│                                      │
│   사진이 전부 가까이에 있는            │
│   것들이거든요.                        │
│   뭐가 끌렸어요?                      │
│                                      │
│   ┌────────────────────────────┐     │
│   │ 여기에 자유롭게 답해주세요   │     │
│   └────────────────────────────┘     │
│                                      │
└──────────────────────────────────────┘
```

에센스 한 줄 32px+ 세리프. 6축은 텍스트 리스트. 팔레트는 사진 추출 5색 원형.
공유 버튼은 이 화면에 없음 — 대화 후 발견이 생긴 뒤 프로필 화면에서 공유.
사용자가 답하면 즉시 `/chat`으로 전환, 첫 메타대화 세션 시작.

## `/home` — 홈

```
┌──────────────────────────────────────┐
│   "손 닿는 곳의 아늑함을              │
│    모으는 사람"          (에센스 한줄) │
│   ─────────────────────               │
│   오늘은 뭘 가져왔어?                 │
│   ┌────────────────────────────┐     │
│   │ 사진, 글, 링크, 메모...          │
│   └────────────────────────────┘     │
│   ─────────────────────               │
│   최근 발견                           │
│   ┌──────────────────────────┐       │
│   │ "닫는 것에 대한 두려움"    │       │
│   │ 3.27 대화에서             │       │
│   └──────────────────────────┘       │
│   ┌──────────────────────────┐       │
│   │ "혼자 채우는 패턴"        │       │
│   │ 3.25 대화에서             │       │
│   └──────────────────────────┘       │
│   ─────────────────────               │
│   ┌──────────────────────────┐       │
│   │  축적 지형도 미니 프리뷰   │       │
│   │         → 전체 보기       │       │
│   └──────────────────────────┘       │
│   ───  ───  ───                      │
│   홈   세션  프로필                    │
└──────────────────────────────────────┘
```

입력하면 즉시 `/chat`으로 전환. 하단 탭 네비게이션.

## `/chat` — 메타대화

```
┌──────────────────────────────────────┐
│  ← 홈                        프로필  │
│                                      │
│   ┌─────────────────────────┐        │
│   │ AI: 왜 풍경만 올렸어?    │        │
│   └─────────────────────────┘        │
│            ┌─────────────────────┐   │
│            │ 나: 색감이 좋아서요  │   │
│            └─────────────────────┘   │
│   ┌─────────────────────────┐        │
│   │ AI: 어떤 색감? 차갑고    │        │
│   │ 넓은 느낌? 따뜻한 빛?   │        │
│   └─────────────────────────┘        │
│   ┌─────────────────────────────┐    │
│   │ 발견: "풍경을 좋아하는 게   │    │
│   │  아니라 넓은 곳이 필요한    │    │
│   │  상태구나"                  │    │
│   │              [ 저장할까요? ] │    │
│   │  ┄ 에센스: 양감 · 온도 ┄   │    │
│   └─────────────────────────────┘    │
│   ┌────────────────────────┐         │
│   │ 메시지 입력           →  │         │
│   └────────────────────────┘         │
└──────────────────────────────────────┘
```

입력 종류별 AI 첫 반응:

| 입력 | 첫 반응 | 탐색 방향 |
|------|---------|----------|
| 풍경 사진 | "왜 풍경만 올렸어?" | 색감/구도/기억 → 현재 상태 연결 |
| 글 링크 | "이 글에서 뭐가 걸렸어?" | 역추적 → 에센스 연결 → 발견 |
| 텍스트 메모 | "이 생각이 왜 지금 떠올랐어?" | 맥락 → 패턴 감지 → 발견 |
| 사물 사진 | "이걸 왜 찍었어?" | 감각 축 분석 → 에센스 연결 |

## `/chat/:sessionId` — 대화 상세

```
┌──────────────────────────────────────┐
│  ← 세션 목록              3.27 (목)  │
│   (해당 세션의 대화 전체 표시)         │
│   ─────────────────────               │
│   이 세션에서 나온 발견               │
│   "풍경을 좋아하는 게 아니라          │
│    넓은 곳이 필요한 상태"             │
│   "닫는 것에 대한 두려움"             │
│   ─────────────────────               │
│   확인된 패턴                         │
│   → "공간 감각과 심리 상태의 연결"    │
└──────────────────────────────────────┘
```

읽기 전용. 대화 원문 + 발견 + 확인된 패턴.

## `/sessions` — 세션 로그

```
┌──────────────────────────────────────┐
│  세션 로그                           │
│   3월                                │
│   3.28 (금) ─────────────────        │
│   풍경 사진 → "넓은 곳이 필요한 상태" │
│   2개 발견                           │
│   3.27 (목) ─────────────────        │
│   글 링크 → "닫는 것에 대한 두려움"   │
│   1개 발견                           │
│   3.25 (화) ─────────────────        │
│   메모 → "혼자 채우는 패턴"           │
│   1개 발견                           │
│   ───  ───  ───                      │
│   홈   세션  프로필                    │
└──────────────────────────────────────┘
```

클릭 시 `/chat/:sessionId`로 이동.

## `/profile` — 프로필

```
┌──────────────────────────────────────┐
│   나의 에센스                         │
│   "손 닿는 곳의 아늑함을              │
│    모으는 사람"                        │
│   양감    채워진 둥근 볼륨             │
│   질감    도자기, 나무결               │
│   투명도  불투명                      │
│   촉각    따뜻하다                    │
│   무게    탁자 위 존재감              │
│   온도    미지근~따뜻                 │
│   ● ● ● ● ●                         │
│   (에센스 변화 시)                    │
│   이전: "채워진 둥근 볼륨"            │
│   현재: "넓고 열린 볼륨"      ← new  │
│   ─────────────────────               │
│   축적 지형도                         │
│   ┌──────────────────────────┐       │
│   │  "닫는 것"  "혼자"  "속도" │       │
│   └──────────────────────────┘       │
│   ─────────────────────               │
│   축적된 발견 (5)                     │
│   "준비 안 된 채로 꺼내는 연습"        │
│   "만드는 건 빠른데 닫는 게 어렵다"    │
│   ...                                │
│   반복 키워드                         │
│   #완성 #혼자 #속도 #팀 #공간        │
│   ───  ───  ───                      │
│   홈   세션  프로필                    │
└──────────────────────────────────────┘
```

지형도: 등고선 스타일, 에센스 팔레트 톤. 클릭 시 해당 영역 인사이트 목록.
