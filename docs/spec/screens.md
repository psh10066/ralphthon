# 랄프톤 — 화면 설계

디자인 톤: 투명, 여백, 콘텐츠 중심. 군더더기 없음.

모바일 퍼스트. 데스크톱 max-width 1200px, 1440px 뷰포트 기준 센터 정렬.

### 글로벌 레이아웃

- 배경: `#EFEEE9` (Warm White) — 전 화면 동일
- **최대 너비: 1200px.** 센터 정렬. fullwidth 금지
- 패딩: 모바일 20px, 태블릿 32px, 데스크톱 40px
- 브레이크포인트: 모바일 < 768px, 태블릿 768~1023px, 데스크톱 ≥ 1024px
- 상단 헤더: droppi 로고 (심볼 + 워드마크) sticky, 배경 blur
- 하단 탭: 홈 / 세션 / 프로필 (3탭). fixed bottom
- 랜딩(`/`)과 읽기 결과(`/result`)에는 하단 탭 없음

## `/` — 랜딩

상단 헤더 없음, 하단 탭 없음 — 풀스크린 센터 레이아웃.

```
┌──────────────────────────────────────┐
│                                      │
│        [심볼 로고 + 워드마크]          │
│           (중앙 배치)                 │
│                                      │
│     뭐든 넣으면                       │
│     당신이 보입니다.                  │
│                                      │
│   ┌────────────────────────────┐     │
│   │                            │     │
│   │   사진 / 글 / 링크 / 메모   │     │
│   │        여기에 drop          │     │
│   │                            │     │
│   └────────────────────────────┘     │
│                                      │
│   ╔══════════════════════════╗       │
│   ║         drop             ║       │
│   ╚══════════════════════════╝       │
│   (bg-[#040000] text-white           │
│    rounded-full px-8 py-3)           │
│                                      │
│  ┌────────┐ ┌────────┐ ┌────────┐   │
│  │주제태그│ │주제태그│ │주제태그│   │
│  │스타일  │ │스타일  │ │스타일  │   │
│  │예시카드│ │예시카드│ │예시카드│   │
│  └────────┘ └────────┘ └────────┘   │
│       ← 가로 스크롤 →                │
└──────────────────────────────────────┘
```

- 로고: 심볼 + 워드마크, 페이지 중앙 상단
- CTA: 블랙 배경 pill 버튼 "drop" (`bg-[#040000] text-white rounded-full px-8 py-3`)
- 예시 카드: 가로 스크롤, 각 카드에 주제/스타일 태그 (컬러 있음)

#### 이미지 자동 리사이즈

```
사진 drop 즉시:
  Canvas API → max 1920px 리사이즈 → JPEG 0.8
  5MB 초과 → quality 0.6 재압축
  그래도 초과 → max 1280px 추가 리사이즈
  HEIC/HEIF → JPEG 변환 (heic2any)
  실패 → 토스트 "이 사진은 올릴 수 없어요"
```

사용자는 리사이즈를 인지하지 못한다. 1장당 < 1초.

#### drop 엣지케이스

| 케이스 | 처리 |
|--------|------|
| GIF 업로드 | 첫 프레임만 추출 |
| 동영상 선택 | 무시 + 토스트 "사진만 올릴 수 있어요" |
| 갤러리 권한 거부 | 토스트 + 설정 딥링크 |
| 오프라인 | 입력은 가능 (로컬). 분석 버튼만 비활성 |
| 빈 텍스트 전송 | drop 버튼 비활성 |

### 로딩

입력 종류별 로딩 카피:

**사진 drop 시**
```
뭐가 찍혀있는지 보는 중
→ 왜 이걸 골랐을지 생각 중
→ 당신의 말로 옮기는 중
```

**글 / 링크 drop 시**
```
어떤 내용인지 읽는 중
→ 뭐가 걸렸을지 생각 중
→ 당신의 말로 옮기는 중
```

drop한 항목 미리보기를 로딩 화면에 함께 표시.

## `/result` — 읽기 결과

상단 헤더 있음 (droppi 로고, sticky blur), 하단 탭 없음.

```
┌──────────────────────────────────────┐
│  [droppi 로고]              (헤더)   │
│──────────────────────────────────────│
│                                      │
│   ┌────────────────────────────┐     │
│   │   (drop한 항목 미리보기)    │     │
│   └────────────────────────────┘     │
│                                      │
│   이렇게 읽어봤어요                   │
│                                      │
│   "손 닿는 곳의 아늑함을              │
│    모으는 사람"                       │
│   (인사이트 — 세리프, 22-28px)        │
│                                      │
│   불투명. 빛을 품는 물질들.            │
│   도자기 유약, 나무결, 따뜻한 온도.    │
│   탁자 위에 올려진 존재감.             │
│   (관찰 사실 목록 — 뮤트 컬러)        │
│                                      │
│   [#공간] [#아늑함] [#물질감]         │
│   (태그 — 컬러 있음)                  │
│                                      │
│   ┌────────────────────────────┐     │
│   │ 뭔가 떠오르면 말해줘요       │     │
│   └────────────────────────────┘     │
│   (자유 입력 필드)                    │
│                                      │
│   [ 공유 ]  [ 저장 ]                  │
│                                      │
│            그냥 넘어갈래요            │
└──────────────────────────────────────┘
```

- 인사이트: 세리프 폰트, 22-28px
- 관찰 사실: 뮤트 컬러 텍스트
- 태그: 컬러 pill 형태
- 자유 입력: 텍스트 있을 때 [→] 활성 → `/chat` 전환
- "그냥 넘어갈래요": 읽기 결과만 저장 후 `/home` 이동

#### 읽기 결과 → 채팅 전환

```
사용자가 자유 입력 필드에 타이핑 + [→] 탭
  ↓
즉시 /chat 으로 전환 (fade 300ms)
  ↓
채팅 화면에 이미 표시됨:
  [읽기 결과 요약 — 상단 고정]
  [사용자] 방금 입력한 반응
  [AI 타이핑 중...]
  ↓
AI 응답 도착 → 바로 표시
  ↓
이후 일반 채팅처럼 자유롭게 주고받기
```

- 입력 필드: 텍스트 비어있으면 [→] 비활성(회색), 입력 있으면 활성
- 전환 후 입력 영역 즉시 활성화
- 읽기 결과 화면의 반응 = 채팅의 첫 번째 사용자 메시지로 세션에 저장

## `/home` — 홈

상단 헤더 (droppi 로고, sticky blur) + 하단 탭 (홈 / 세션 / 프로필).

```
┌──────────────────────────────────────┐
│  [droppi 로고]              (헤더)   │
│──────────────────────────────────────│
│                                      │
│   "손 닿는 곳의 아늑함을              │
│    모으는 사람"          (에센스 한줄) │
│   ─────────────────────               │
│   오늘은 뭘 가져왔어?                 │
│   ┌────────────────────────────────┐ │
│   │ White 배경                     │ │
│   │ 사진, 글, 링크, 메모 drop       │ │
│   │                        drop    │ │  ← 우측에 "drop" 밑줄 텍스트
│   └────────────────────────────────┘ │
│   ─────────────────────               │
│   최근 읽기                           │
│   ┌──────────────────────────┐       │
│   │ (읽기 결과 카드 1)        │       │  ← 세션 페이지와 동일 카드 스타일
│   └──────────────────────────┘       │
│   ┌──────────────────────────┐       │
│   │ (읽기 결과 카드 2)        │       │
│   └──────────────────────────┘       │
│   ─────────────────────               │
│   주제 모음                           │
│   ● 공간 4  ● 혼자 3  ● 속도 2  ...  │  ← 컬러 dot + 라벨 + 개수
│   ───  ───  ───                      │
│   홈   세션  프로필                    │
└──────────────────────────────────────┘
```

- drop 입력 영역: White 배경 카드, 우측에 "drop" 밑줄 텍스트 버튼
- 최근 읽기: 최근 3-5개만 프리뷰, 카드 스타일은 세션 페이지와 동일
- 주제 모음: 하단에 칩 형태 프리뷰 (컬러 dot + 라벨 + 개수)
- 입력하면 즉시 로딩 → `/result` 흐름

## `/sessions` — 세션

모든 읽기를 시간순으로 보여주는 전체 목록.

상단 헤더 (droppi 로고, sticky blur) + 하단 탭 (홈 / 세션 / 프로필).

```
┌──────────────────────────────────────┐
│  [droppi 로고]              (헤더)   │
│──────────────────────────────────────│
│                                      │
│  ┌─────────────────────────────────┐ │
│  │ [사진/컬러│ 타입 라벨            │ │
│  │ 배경+날짜│ "인사이트 첫 줄..."   │ │
│  │ 오버레이 │ 관찰 사실 (뮤트)      │ │
│  │ (100px) │ [#태그] [#태그]       │ │
│  └─────────────────────────────────┘ │
│  ┌─────────────────────────────────┐ │
│  │ [컬러배경│ 타입 라벨            │ │
│  │ + 날짜  │ "인사이트 첫 줄..."   │ │
│  │ 큰텍스트│ 관찰 사실 (뮤트)      │ │
│  │ (100px) │ [#태그] [#태그]       │ │
│  └─────────────────────────────────┘ │
│  ...                                 │
│   ───  ───  ───                      │
│   홈   세션  프로필                    │
└──────────────────────────────────────┘
```

매거진 그리드 레이아웃:
- 모바일: 1열, 데스크톱: 2열
- 각 카드 높이 통일 (160px)
- 왼쪽 영역 (100-120px): 사진이 있으면 사진 + 날짜 오버레이, 없으면 주제 컬러 배경 + 날짜 큰 텍스트
- 오른쪽 영역: 타입 라벨 → 인사이트 (세리프, line-clamp-2) → 관찰 사실 (뮤트) → 태그 칩

하단 탭에서 "세션" 탭으로 진입.

## `/chat` — 메타대화

```
┌──────────────────────────────────────┐
│  ← 홈                        프로필  │
│                                      │
│   ┌────────────────────────────┐     │
│   │   (읽기 결과 요약 — 상단)   │     │
│   └────────────────────────────┘     │
│                                      │
│   ┌─────────────────────────┐        │
│   │ AI: 이게 지금 떠오른 게   │        │
│   │ 신기하지 않아요?          │        │
│   └─────────────────────────┘        │
│            ┌─────────────────────┐   │
│            │ 나: 그러게요, 사실   │   │
│            └─────────────────────┘   │
│   ┌─────────────────────────────┐    │
│   │ 발견: "풍경을 좋아하는 게   │    │
│   │  아니라 넓은 곳이 필요한    │    │
│   │  상태구나"                  │    │
│   │              [ 저장할까요? ] │    │
│   │  ┄ 인사이트 블록 인라인 ┄   │    │
│   └─────────────────────────────┘    │
│   ┌────────────────────────┐         │
│   │ 메시지 입력           →  │         │
│   └────────────────────────┘         │
└──────────────────────────────────────┘
```

진입 경로: 읽기 결과 화면에서 사용자가 응답을 입력했을 때.
읽기 결과가 대화 상단에 고정 표시. 인사이트 블록이 대화 흐름 안에 인라인으로 나타남.

#### 채팅 레이아웃 — 고정 입력, 스크롤 메시지

**핵심: 새 메시지가 아래에 추가되고, 자동 스크롤.**

```
┌──────────────────────────────────────┐
│  ← 홈                                │  ← 헤더 (고정)
│  ┌─── 스크롤 영역 ──────────────┐    │
│  │  [읽기 결과 요약 — 상단]       │    │
│  │  [사용자 반응]                 │    │
│  │  [AI 메시지]                  │    │
│  │  [인사이트 카드]               │    │  ← 최신 메시지가 항상 보임
│  │  [AI 타이핑 중...]            │    │
│  └───────────────────────────────┘    │
│  ┌────────────────────────────┐      │  ← 입력 영역 (키보드 위 고정)
│  │ 메시지 입력           📷 [→] │      │
│  └────────────────────────────┘      │
└──────────────────────────────────────┘
```

**채팅 중에는 하단 탭바를 숨긴다.** 나가려면 헤더 `← 홈` 사용.

#### 스크롤 규칙

| 상황 | 동작 |
|------|------|
| 새 AI 메시지 도착 (스크롤이 맨 아래) | 자동 스크롤하여 새 메시지 표시 |
| 사용자가 위로 스크롤한 상태에서 새 메시지 | 자동 스크롤 안 함. "↓ 새 메시지" 플로팅 버튼 |
| 사용자 메시지 전송 | 항상 맨 아래로 스크롤 |
| 키보드 올라올 때 | 마지막 메시지가 보이도록 스크롤 조정 |

#### 전송 후 상태 리셋

```typescript
const sendMessage = async (content, type) => {
  try {
    setIsSending(true);
    // API 호출
  } catch (error) {
    // 에러 처리
  } finally {
    setIsSending(false);
    setAttachedPhotos([]);
    inputRef.current?.focus();
  }
};
```

- 전송 후 동일하게 상태 리셋
- AI 응답 대기 중: 입력은 가능하지만 전송은 불가 (`[→]` 비활성)
- AI 응답 도착 후: `[→]` 즉시 활성화

#### 세션 유지

| 시나리오 | 동작 |
|---------|------|
| 채팅 중 헤더 `← 홈` 탭 | `/home`으로 이동. 세션 저장됨 |
| `/home`에서 다시 채팅 시작 | 24시간 이내: "이어서 이야기할까요?" 배너 표시 |
| 브라우저 닫기 → 재방문 | 마지막 세션 24시간 이내면 이어서 가능 |

채팅 중 탭바가 숨겨지므로 "탭 이동 후 세션 사라짐" 문제 자체가 발생하지 않음.

#### 입력 종류별 AI 첫 반응

| 입력 | 첫 반응 | 탐색 방향 |
|------|---------|----------|
| 풍경 사진 | "왜 풍경만 올렸어?" | 색감/구도/기억 → 현재 상태 연결 |
| 글 링크 | "이 글에서 뭐가 걸렸어?" | 역추적 → 에센스 연결 → 발견 |
| 텍스트 메모 | "이 생각이 왜 지금 떠올랐어?" | 맥락 → 패턴 감지 → 발견 |
| 사물 사진 | "이걸 왜 찍었어?" | 감각 축 분석 → 에센스 연결 |

#### 채팅 엣지케이스

| 케이스 | 처리 |
|--------|------|
| 빈 메시지 전송 | `[→]` 비활성이므로 불가능 |
| 2000자 초과 | 제한 + 카운터 표시 |
| AI 응답 파싱 실패 | "다시 생각해볼게요..." → 자동 재시도 1회 |
| 짧은 응답 ("응", "ㅇㅇ") | AI가 구체적 질문: "좀 더 이야기해줄 수 있어요?" |
| 네트워크 끊김 중 전송 | 로컬 큐 저장 + 빨간 `!` 표시. 탭 시 재전송 |

```
│  │ 공간                         │    │
│  │ 읽기 4개 · 대화 2개           │    │
│  └──────────────────────────────┘    │
│           ↕ margin-top: -24px        │
│  ████████████████████████████████    │ ← 혼자 컬러 바 (5px)
│  ┌──────────────────────────────┐    │
│  │ 혼자                         │    │
│  │ 읽기 3개 · 대화 1개           │    │
│  └──────────────────────────────┘    │
│           ↕ margin-top: -24px        │
│  ████████████████████████████████    │ ← 속도 컬러 바 (5px)
│  ┌──────────────────────────────┐    │
│  │ 속도                         │    │
│  │ 읽기 2개 · 대화 0개           │    │
│  └──────────────────────────────┘    │
│                                      │
│   ───  ───  ───                      │
│   홈   세션  프로필                    │
└──────────────────────────────────────┘
```

- 카드 상단에 주제 컬러 바 (5px)
- 카드들은 `margin-top: -24px`으로 겹쳐 쌓임
- 탭하면 해당 카드가 펼쳐지고 아래 카드들이 밀려남
- 펼쳐진 카드 안에 해당 주제의 읽기 목록 인라인 표시
- 각 테마 카드 탭 → 인라인 펼침 또는 `/themes/:themeId`로 이동

빈 상태: "아직 주제가 없어요. 뭐든 drop하면 비슷한 것끼리 묶여요."

## `/profile` — 프로필

상단 헤더 (droppi 로고, sticky blur) + 하단 탭 (홈 / 세션 / 프로필).

```
┌──────────────────────────────────────┐
│  [droppi 로고]              (헤더)   │
│──────────────────────────────────────│
│                                      │
│   나의 에센스                         │
│   "손 닿는 곳의 아늑함을              │
│    모으는 사람"    (세리프 폰트)       │
│                                      │
│   ● ● ● ● ●  (컬러 팔레트)          │
│                                      │
│   ─────────────────────               │
│   이번 달 흐름                        │
│   주제 분포 ████░░░ 공간             │
│            █████░░ 혼자             │
│            ███░░░░ 속도             │
│   스타일 분포 ██████░ 차분한          │
│              ████░░░ 관조적          │
│   지난달엔 '아늑함'이 많았는데,        │
│   이번 달은 '경계' 쪽으로 가고 있어요. │
│   ─────────────────────               │
│   축적된 발견 (5)                     │
│   "준비 안 된 채로 꺼내는 연습"        │
│   "만드는 건 빠른데 닫는 게 어렵다"    │
│   ...                                │
│   ─────────────────────               │
│   상위 태그                          │
│   #공간 #혼자 #속도 #팀              │
│   #차분한 #관조적 #느린              │
│   ───  ───  ───                      │
│   홈   세션  프로필                    │
└──────────────────────────────────────┘
```

- 에센스 한줄: 세리프 폰트
- 팔레트: 컬러 dot 5개
- 이번 달 흐름: 주제 분포 바 + 스타일 분포 바 + 이전 달 비교 텍스트
- 발견 목록: 축적된 발견 전체 (날짜 표시)
- 상위 태그: 주제 태그 + 스타일 태그 통합 표시
