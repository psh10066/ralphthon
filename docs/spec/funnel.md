# 랄프톤 — 퍼널

---

## 퍼널 흐름도

```
[1] 랜딩         호기심만. 뭐든 drop하게 만든다.
        ↓
[2] 로딩         "읽고 있어요..." → 입력별 카피
        ↓
[3] 읽기 결과     인사이트 한 덩어리 + 관찰. 대화는 선택  ← 핵심
        ↓
[4] 홈           일상 진입점. drop + 최근 읽기 + 주제 모음
        ↓
[5] 축적          프로필 + 주제 모음 + 발견
```

---

## 각 섹션 카피

### [1] 랜딩

> 뭐든 넣으면 당신이 보입니다.

- 흰 배경, 큰 세리프, 여백만
- 설명 없음. 호기심만 건드린다
- drop 영역: 사진, 글, 링크, 메모 아무거나 올려두는 곳
- 예시 읽기 결과 카드 3개 가로 스크롤

### [2] 로딩

> 읽고 있어요...

입력 종류에 따라 카피가 달라진다.
- 사진 drop 시: **이미지 자동 리사이즈** (max 1920px, JPEG 0.8, HEIC→JPEG 변환. 사용자 인지 없이 처리)

| 입력 종류 | 로딩 카피 |
|-----------|-----------|
| 사진 | "뭐가 찍혀있는지 보는 중 → 왜 이걸 골랐을지 생각 중 → 당신의 말로 옮기는 중" |
| 글/링크 | "어떤 내용인지 읽는 중 → 뭐가 걸렸을지 생각 중 → 당신의 말로 옮기는 중" |

완료 = 체크, 현재 = 채워진 원, 대기 = 빈 원. 부드러운 전환 애니메이션.

### [3] 읽기 결과 (wow point → 첫 대화)

> "손 닿는 곳의 아늑함을 모으는 사람"

- "이렇게 읽어봤어요" — 겸손한 톤으로 시작
- 읽기 결과 인사이트 한 덩어리가 가장 크게 (32px+, 세리프)
- 6축 관찰 텍스트 리스트: 양감 / 질감 / 투명도 / 촉각 / 무게 / 온도 — wow point가 아닌 보조 정보
- "이게 다는 아닐 수 있어요. 더 알고 싶어서 하나 물어볼게요."
- 자유 입력 필드: 생각나는 것, 반응, 무엇이든 적을 수 있다
- 스킵 옵션: 지금은 넘어가고 싶으면 바로 홈으로 (읽기만 저장)
- **답하면 즉시 `/chat`으로 전환** — 추가 전송 없이 바로 채팅 시작
- 전환 후 채팅 화면에: 읽기 결과 요약 + 사용자 반응 + AI 응답(자동 호출)
- 공유 / 저장 버튼 있음

### [4] 홈

> 오늘은 뭘 가져왔어?

- 에센스 한 줄이 항상 상단
- 입력 영역: 사진, 텍스트, 링크, 메모 모두 drop 가능
- 입력하면 로딩 → 읽기 결과 흐름. 읽기 결과에서 반응하면 `/chat`으로 전환
- 최근 읽기 카드 + 주제 모음 칩

### [5] 축적

프로필에서 에센스 + 주제 모음 + 발견 목록 + 반복 키워드를 한눈에 본다.
6축은 프로필에서 collapsible로 표시.
주제 모음에서 주제별로 묶인 읽기 클러스터를 탐색할 수 있다.

## 랜딩 예시 카드 3개

### 카드 1

```
"손 닿는 곳의 아늑함을 모으는 사람"
읽기 결과: 따뜻하고 둥근 것들에 반복적으로 이끌린다
팔레트: #C4A882, #D4A5A5, #8FA68E
```

### 카드 2

```
"시원하게 비어있는 곳에서 쉬는 사람"
읽기 결과: 비움과 열린 공간에서 안도감을 찾는다
팔레트: #E8EBE4, #9BB5C4, #F5F5F5
```

### 카드 3

```
"묵직하게 쌓아올리는 것에 끌리는 사람"
읽기 결과: 단단하고 무게감 있는 것들이 반복해서 등장한다
팔레트: #4A4A4A, #8B6F47, #2C2C2C
```

카드 클릭 시 읽기 결과 상세 모달. 모달 하단 "나도 해볼래" CTA.
