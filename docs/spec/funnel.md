# 랄프톤 — 퍼널

---

## 퍼널 흐름도

```
[1] 랜딩         호기심만. 설명 없음. 바로 업로드.
        ↓
[2] 업로드        끌리는 사진 5장
        ↓
[3] 로딩         "색감을 보고 있어요..." → "질감을 느끼고 있어요..." → ...
        ↓
[4] 거울          에센스 한 줄 + 6축 + 컬러팔레트 + 가설 질문  ← wow point
        ↓
[5] 홈           일상 진입점. 뭐든 입력 → 즉시 대화
        ↓
[6] 축적          프로필 + 지형도 + 세션 로그
```

---

## 각 섹션 카피

### [1] 랜딩

> 사진 5장이면 당신이 보입니다.

- 흰 배경, 큰 세리프, 여백만
- 설명 없음. 호기심만 건드린다
- 예시 에센스 카드 3개 가로 스크롤
- CTA: `[ 시작하기 ]`

### [2] 업로드

> 끌리는 사진으로. 예쁠 필요 없어요.

- **갤러리에서 최대 5장 한꺼번에 선택** (`<input multiple>`) — 하나씩 반복 금지
- 5장 채워야 "나를 읽어줘" 활성화 (미만: 회색 비활성)
- 업로드 시 투명한 유리 오버레이
- 모바일: 카메라롤 직접 접근
- **이미지 자동 리사이즈**: max 1920px, JPEG 0.8, HEIC→JPEG 변환. 사용자 인지 없이 처리
- 개별 사진 `[ × ]`로 제거 가능, 길게 누르기로 순서 변경
- CTA: `[ 나를 읽어줘 ]`

### [3] 로딩

> 당신을 읽고 있어요...

4단계 순차 진행. 완료 = 체크, 현재 = 채워진 원, 대기 = 빈 원. 부드러운 전환 애니메이션.

### [4] 거울 (wow point → 첫 대화)

> "손 닿는 곳의 아늑함을 모으는 사람"

- "사진에서 이렇게 읽어봤어요" — 겸손한 톤으로 시작
- 에센스 한 줄이 가장 크게 (32px+, 세리프)
- 6축은 텍스트 리스트로 가볍게 (바 차트 아님)
- 컬러 팔레트: 사진에서 추출한 5색, 작은 원형
- "이게 다는 아닐 수 있어요. 더 알고 싶어서 하나 물어볼게요."
- 관찰 사실 + 열린 질문 (가설이 아니라 호기심)
- **`[답하기]` 버튼**: 입력 비어있으면 비활성(회색). 입력 있으면 활성
- **답하면 즉시 `/chat`으로 전환** — 추가 전송 없이 바로 채팅 시작 (BUG-003)
- 전환 후 채팅 화면에: AI 첫 턴 + 사용자 응답 + AI 두 번째 턴(자동 호출)
- 공유는 이 화면에서 하지 않음 (대화 후 발견이 생기면 그때)

### [5] 홈

> 오늘은 뭘 가져왔어?

- 에센스 한 줄이 항상 상단
- 입력 영역: 사진, 텍스트, 링크, 메모 모두 가능
- 입력하면 즉시 `/chat`으로 전환, 메타대화 시작
- 최근 발견 카드 + 축적 지형도 미니 프리뷰

### [6] 축적

프로필에서 에센스 + 지형도 + 발견 목록 + 반복 키워드를 한눈에 본다.
세션 로그에서 날짜별 대화 기록과 발견을 다시 확인할 수 있다.

| 순서 | 텍스트 | 시간 |
|------|--------|------|
| 1 | "색감을 보고 있어요" | 3초 |
| 2 | "질감을 느끼고 있어요" | 3초 |
| 3 | "온도를 재고 있어요" | 3초 |
| 4 | "언어로 옮기고 있어요" | 3초 |

총 ~12초. API 응답 시간에 맞춤. 기대감을 쌓는 장치이기도 하다.

## 랜딩 예시 카드 3개

### 카드 1

```
"손 닿는 곳의 아늑함을 모으는 사람"
양감: 채워진 둥근 볼륨 | 질감: 도자기, 나무결
팔레트: #C4A882, #D4A5A5, #8FA68E
```

### 카드 2

```
"시원하게 비어있는 곳에서 쉬는 사람"

팔레트: #E8EBE4, #9BB5C4, #F5F5F5
```

### 카드 3

```
"묵직하게 쌓아올리는 것에 끌리는 사람"
양감: 무겁고 견고한 | 질감: 콘크리트, 가죽, 짙은 나무
팔레트: #4A4A4A, #8B6F47, #2C2C2C
```

카드 클릭 시 에센스 상세 모달 (6축 + 가설 질문). 모달 하단 "나도 해볼래" CTA.
